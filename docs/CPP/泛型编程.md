## 函数模板



函数模板

```cpp
template <typename T>
T max(T a, T b) {
    return a > b ? a : b;
}
```

隐式实例化

```cpp
// 编译时编译器按需生成
int max(int a, int b) {
    return a > b ? a : b;
}

int main() {
    int a = 10, b = 30;
    int c = max<int>(a, b);
    return 0;
}
```

显式实例化

```cpp
template float max<float>(float a, float b);
```

## 类模板

```cpp
template <typename T>
class Vector3 {
public:
    Vector3(T v1, T v2, T v3) {
        m_vec[0] = v1;
        m_vec[1] = v2;
        m_vec[2] = v3;
    }
    T getMax();
private:
    T m_vec[3];
};

template <typename T>
T Vector3<T>::getMax() {
    T temp = m_vec[0] > m_vec[1] ? m_vec[0] : m_vec[1];
    return temp > m_vec[2] ? temp : m_vec[2];
}
```



模板中还可以添加默认参数：

```cpp
template <typename T, int N = 10>
class Vector3 {
public:
    Vector3(T v1, T v2, T v3) : maxNums(N) {
        m_vec[0] = v1;
        m_vec[1] = v2;
        m_vec[2] = v3;
    }
    T getMax();
private:
    T m_vec[3];
    int maxNums;
};
```
