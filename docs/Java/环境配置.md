### 配置 VS Code 风格快捷键

（1）安装 VS Code Keymap 插件

打开 IDEA → `File` → `Settings` (Windows/Linux: `Ctrl+Alt+S` / Mac: `Cmd+,`)

进入 `Plugins` → `Marketplace`

搜索：**VSCode Keymap** → 安装

（2）切换到 VS Code 键位

进入 `Settings` → `Keymap`

顶部下拉选择：**VSCode** → `Apply` → `OK`
