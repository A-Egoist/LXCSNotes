    add_executable ж·»еҠ дёҖдёӘеҸҜжү§иЎҢж–Үд»¶зӣ®ж ҮгҖӮжҢҮе®ҡзӣ®ж ҮеҗҚз§°е’ҢжүҖжңүжәҗж–Үд»¶гҖӮ
    add_library ж·»еҠ дёҖдёӘеә“зӣ®ж ҮгҖӮеҸҜд»ҘжҢҮе®ҡдёә STATICпјҲйқҷжҖҒеә“пјүжҲ– SHAREDпјҲеҠЁжҖҒеә“пјүгҖӮ
    target_link_libraries е°ҶдёҖдёӘзӣ®ж ҮпјҲдҫӢеҰӮеҸҜжү§иЎҢж–Үд»¶пјүдёҺжүҖйңҖзҡ„еә“й“ҫжҺҘиө·жқҘгҖӮ
    set и®ҫзҪ®дёҖдёӘеҸҳйҮҸгҖӮ
    include_directories дёәзј–иҜ‘иҝҮзЁӢж·»еҠ еӨҙж–Үд»¶жҗңзҙўи·Ҝеҫ„гҖӮ
    add_subdirectory еҢ…еҗ«еӯҗзӣ®еҪ•дёӯзҡ„ CMakeLists.txt ж–Үд»¶гҖӮиҝҷжҳҜз»„з»ҮеӨ§еһӢйЎ№зӣ®зҡ„йҮҚиҰҒж–№ејҸгҖӮ

    жӣҙеӨҚжқӮзҡ„йЎ№зӣ®зӨәдҫӢ¶

    йЎ№зӣ®еңәжҷҜ¶

    • calculator зЁӢеәҸпјҡи°ғз”Ё math_lib дёӯзҡ„ add еҮҪж•°гҖӮ

    • add_subdirectory()пјҡе°ҶеӨ§еһӢйЎ№зӣ®еҲҶи§ЈжҲҗжӣҙе°Ҹзҡ„гҖҒеҸҜз®ЎзҗҶзҡ„еҚ•е…ғгҖӮ
    • add_library() / add_executable()пјҡе®ҡд№үжһ„е»әзӣ®ж ҮгҖӮ
    • target_include_directories() / target_link_libraries()пјҡз®ЎзҗҶзӣ®ж Үзҡ„дҫқиө–е…ізі»пјҲеӨҙж–Үд»¶е’Ңеә“пјүгҖӮ

    иҝҷдәӣе‘Ҫд»Өжһ„жҲҗдәҶеӨ§еӨҡж•° CMake йЎ№зӣ®зҡ„еҹәзЎҖпјҢзҶҹз»ғжҺҢжҸЎе®ғд»¬е°ҶеӨ§еӨ§з®ҖеҢ–дҪ зҡ„йЎ№зӣ®з®ЎзҗҶе·ҘдҪңгҖӮ

    иҝҷйҮҢзҡ„ -D йҖүйЎ№з”ЁжқҘе®ҡд№үдёҖдёӘ CMake еҸҳйҮҸгҖӮCMAKE_BUILD_TYPE жҳҜдёҖдёӘеҶ…зҪ®еҸҳйҮҸпјҢеҸҜд»Ҙи®ҫзҪ®д»ҘдёӢеҮ дёӘеёёи§Ғзҡ„еҖјпјҡ

    • Debugпјҡз”ҹжҲҗеёҰи°ғиҜ•дҝЎжҒҜзҡ„д»Јз ҒпјҲзӯүеҗҢдәҺ g++ -gпјүпјҢйҖҡеёёдёҚдјҡејҖеҗҜд»Јз ҒдјҳеҢ–пјҢйҖӮеҗҲ GDB и°ғиҜ•гҖӮ
    • Releaseпјҡз”ҹжҲҗдјҳеҢ–иҝҮзҡ„д»Јз ҒпјҲзӯүеҗҢдәҺ g++ -O3пјүпјҢдёҚеҢ…еҗ«и°ғиҜ•дҝЎжҒҜпјҢйҖӮеҗҲжңҖз»ҲеҸ‘еёғз»ҷз”ЁжҲ·зҡ„зүҲжң¬гҖӮ
    • RelWithDebInfoпјҡз”ҹжҲҗдјҳеҢ–иҝҮзҡ„д»Јз ҒпјҢдҪҶдҝқз•ҷи°ғиҜ•дҝЎжҒҜпјҢз”ЁдәҺйңҖиҰҒи°ғиҜ• Release зүҲжң¬жҖ§иғҪй—®йўҳгҖӮ
    • MinSizeRelпјҡз”ҹжҲҗдёәеҮҸе°‘д»Јз ҒдҪ“з§ҜиҖҢдјҳеҢ–зҡ„д»Јз ҒгҖӮ
